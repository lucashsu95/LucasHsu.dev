Github

йҖҷжҳҜдёҖеҖӢз”ЁдҫҶжү№ж¬ЎиӘҝж•ҙең–зүҮеӨ§е°Ҹзҡ„ Python зЁӢејҸгҖӮ

еҠҹиғҪ вҖӢ

йҖҷеҖӢзЁӢејҸеҸҜд»ҘиҮӘеӢ•иӘҝж•ҙжҢҮе®ҡиіҮж–ҷеӨҫдёӯзҡ„жүҖжңүең–зүҮеӨ§е°ҸпјҢдҪҝе…¶еҜ¬еәҰдёҖиҮҙпјҢеҗҢжҷӮдҝқжҢҒеҺҹе§Ӣең–зүҮзҡ„еҜ¬й«ҳжҜ”гҖӮдё»иҰҒеҠҹиғҪеҢ…жӢ¬пјҡ

  • жү№ж¬Ўиҷ•зҗҶеӨҡеҖӢең–зүҮ
  • зўәдҝқијёеҮәиіҮж–ҷеӨҫеӯҳеңЁ
  • иҮӘеӢ•иЁҲз®—зӯүжҜ”дҫӢй«ҳеәҰ
  • иӘҝж•ҙең–зүҮи§ЈжһҗеәҰ
  • иҷ•зҗҶд»»дҪ•йҢҜиӘӨдёҰжҸҗдҫӣйҢҜиӘӨиЁҠжҒҜ

дҪҝз”Ёж–№ејҸ вҖӢ

  1. е°ҮиҰҒиӘҝж•ҙеӨ§е°Ҹзҡ„ең–зүҮж”ҫе…Ҙијёе…ҘиіҮж–ҷеӨҫдёӯгҖӮ
  2. еңЁзЁӢејҸдёӯжҢҮе®ҡијёе…ҘиіҮж–ҷеӨҫзҡ„и·Ҝеҫ‘пјҲinput_folderпјүе’ҢијёеҮәиіҮж–ҷеӨҫзҡ„и·Ҝеҫ‘пјҲoutput_folderпјүгҖӮ
  3. еҹ·иЎҢзЁӢејҸпјҢиӘҝж•ҙеҫҢзҡ„ең–зүҮе°ҮжңғдҝқеӯҳеңЁијёеҮәиіҮж–ҷеӨҫдёӯгҖӮ

еҰӮдҪ•дҪҝз”Ё вҖӢ

еҹ·иЎҢзЁӢејҸ вҖӢ

bash
python resize_images.py

и«Ӣж №ж“ҡдҪ зҡ„йңҖжұӮдҝ®ж”№ input_folder е’Ң output_folder зҡ„и·Ҝеҫ‘гҖӮ

жіЁж„ҸдәӢй … вҖӢ

  • и«Ӣзўәдҝқијёе…ҘиіҮж–ҷеӨҫдёӯеҸӘеҢ…еҗ«ең–зүҮжӘ”жЎҲпјҢеҗҰеүҮзЁӢејҸеҸҜиғҪжңғеҮәзҸҫйҢҜиӘӨгҖӮ
  • и«ӢжӘўжҹҘзЁӢејҸеҹ·иЎҢеҫҢзҡ„ијёеҮәиіҮж–ҷеӨҫпјҢзўәдҝқиӘҝж•ҙеҫҢзҡ„ең–зүҮз¬ҰеҗҲй җжңҹгҖӮ
