氣泡排序法 ​

TA001 排序練習 氣泡排序法 ​

python
import sys

for line in sys.stdin.read().splitlines():
    nums = list(map(int, line.split(", ")))
    n = len(nums)
    for i in range(n - 1):
        for j in range(n - i - 1):
            if nums[j] > nums[j + 1]:
                nums[j], nums[j + 1] = nums[j + 1], nums[j]
        if i == 1:
            break
    print(nums)
