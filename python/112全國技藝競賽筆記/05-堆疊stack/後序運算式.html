# йЎҢзӣ® 2пјҡеҫҢеәҸ(Postfix)иҪүжҸӣе•ҸйЎҢ (10%) (дёӯеәҸиҪүеҫҢеәҸ)
st = input()
level = {'+': 1, '-': 1, '*': 2, '/': 2}

output = []
stack = []

for char in st:
    if char.isalnum():
        output.append(char)
    elif char in '+-*/':

        while (stack and (stack[-1] in '+-*/') and (level[stack[-1]] >= level[char])):
            output.append(stack.pop())
        stack.append(char)

    elif char == '(':
        stack.append(char)
    elif char == ')':  # йҒҮеҲ°жӢ¬иҷҹ
        while stack and stack[-1] != '(':
            output.append(stack.pop())
        stack.pop()
    print(stack,output)

while stack:
    output.append(stack.pop())

print(''.join(output))

data = list(input().split())
stack = []
while data:
    num_or_op = data.pop(0)
    try:    #int
        eval(num_or_op)
        stack.append(num_or_op)
    except: #op
        b = stack.pop()
        a = stack.pop()
        stack.append(str(eval(a+num_or_op+b)))
print(f'{float(stack[0]):.0f}')
