зЁӢејҸиЁӯиЁҲеё«зҡ„з”ҹзү©йҗҳеҸӘжңүе…©еҖӢзӢҖж…Ӣпјҡcodingе’ҢйӮ„жІ’еҜ«е®Ңзҡ„codingгҖӮ

еҲ—иЎЁжҳҜ Python дёӯеёёз”Ёзҡ„иіҮж–ҷзөҗж§ӢпјҢз”Ёж–је„ІеӯҳдёҖзі»еҲ—е…ғзҙ гҖӮе®ғе…·жңүеҪҲжҖ§пјҢеҸҜд»Ҙе®№зҙҚдёҚеҗҢйЎһеһӢзҡ„е…ғзҙ пјҢеҰӮж•ёеӯ—гҖҒеӯ—дёІгҖҒзү©д»¶зӯүгҖӮ

е»әз«ӢеҲ—иЎЁ вҖӢ

жӮЁеҸҜд»ҘдҪҝз”Ёд»ҘдёӢж–№ејҸе»әз«ӢдёҖеҖӢеҲ—иЎЁпјҡ

py
my_list = [е…ғзҙ 1, е…ғзҙ 2, е…ғзҙ 3]

еңЁйҖҷиЈЎпјҢжӮЁеҸҜд»Ҙе°Үд»»дҪ•жғіиҰҒе„Іеӯҳзҡ„е…ғзҙ ж”ҫе…Ҙж–№жӢ¬иҷҹе…§пјҢд»ҘйҖ—иҷҹеҲҶйҡ”гҖӮ

иЁӘе•Ҹе…ғзҙ  вҖӢ

еҲ—иЎЁдёӯзҡ„е…ғзҙ еҸҜд»ҘйҖҡйҒҺзҙўеј•иЁӘе•ҸпјҢзҙўеј•еҫһ 0 й–Ӣе§ӢгҖӮдҫӢеҰӮпјҡ

py
first_element = my_list[0]

жӮЁйӮ„еҸҜд»ҘдҪҝз”ЁеҲҮзүҮдҫҶзҚІеҸ–еӯҗеҲ—иЎЁпјҡ

py
sub_list = my_list[1:]

еҲ—иЎЁж–№жі• вҖӢ

Python жҸҗдҫӣдәҶеӨҡзЁ®ж–№жі•дҫҶж“ҚдҪңеҲ—иЎЁгҖӮд»ҘдёӢжҳҜдёҖдәӣеёёз”Ёзҡ„ж–№жі•зӨәдҫӢпјҡ

  • append ж–°еўһ вҖӢ

    е°Үж–°е…ғзҙ еўһеҠ еҲ°еҲ—иЎЁжң«е°ҫгҖӮ

    py
    my_list.append(ж–°е…ғзҙ )

  • remove еҲӘйҷӨ вҖӢ

    еҲӘйҷӨзү№е®ҡзҡ„е…ғзҙ гҖӮ

    py
    my_list.remove(иҰҒеҲӘйҷӨзҡ„е…ғзҙ )

  • insert жҸ’е…Ҙ вҖӢ

    еңЁжҢҮе®ҡдҪҚзҪ®жҸ’е…Ҙж–°е…ғзҙ гҖӮ

    py
    my_list.insert(дҪҚзҪ®, ж–°е…ғзҙ )

  • copy иӨҮиЈҪ вҖӢ

    иӨҮиЈҪиҲҠйҷЈеҲ—еҲ°ж–°йҷЈеҲ—пјҢдёҚжңғзӣёдә’е№Іж“ҫ

    py
    my_list2 = my_list.copy()
my_list3 = my_list[:] # жңүеҗҢжЁЈж Ўжһң

    еҲӘйҷӨзү№е®ҡзҡ„дҪҚзҪ®зҡ„е…ғзҙ пјҢдёҰеӣһеӮігҖӮ

    py
    item = my_list.pop(дҪҚзҪ®) # еӣһеӮіиў«еҲӘйҷӨзҡ„е…ғзҙ 

  • index зҙўеј• вҖӢ

    еӣһеӮіжҢҮе®ҡе…ғзҙ дҪҚзҪ®гҖӮ

    py
    idx = my_list.index(е…ғзҙ ) # еӣһеӮіе…ғзҙ зҡ„дҪҚзҪ®

    my_list.sort()

    my_list.reverse()

  • append ж–°еўһ вҖӢ

  • insert жҸ’е…Ҙ вҖӢ

    еңЁжҢҮе®ҡдҪҚзҪ®жҸ’е…Ҙж–°е…ғзҙ гҖӮ

    py

  • copy иӨҮиЈҪ вҖӢ

    иӨҮиЈҪиҲҠйҷЈеҲ—еҲ°ж–°йҷЈеҲ—пјҢдёҚжңғзӣёдә’е№Іж“ҫ
