# TQC+ 程式語言Python 107 數值計算
題目說明：請撰寫一程式，讓使用者輸入五個數字，計算並輸出這五個數字之數值、總和及平均數。
> 提示：總和與平均數皆輸出到小數點後第1位。
### 範例輸入1
```py
20
40
60
80
100
```
### 範例輸出1
```py
20 40 60 80 100
Sum = 300.0
Average = 60.0
```
### 範例輸入2
```py
88.7
12
56
132.55
3
```
### 範例輸出2
```py
88.7 12 56 132.55 3
Sum = 292.2
Average = 58.5
```
