# TQC+ 程式語言Python 209 距離判斷
設計說明：請使用選擇敘述撰寫一程式，讓使用者輸入一個點的平面座標x和y值，判斷此點是否與點(5, 6)的距離小於或等於15，如距離小於或等於15顯示【Inside】，反之顯示【Outside】。
> 提示：計算平面上兩點距離的公式：$\sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$。
### 範例輸入1
```shell
7
20
```
### 範例輸出1
```shell
Inside
```
### 範例輸入2
```shell
30
35
```
### 範例輸出2
```shell
Outside
```
