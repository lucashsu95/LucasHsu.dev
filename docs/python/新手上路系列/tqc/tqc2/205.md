# TQC+ 程式語言Python 205 字元判斷
題目說明：請使用選擇敘述撰寫一程式，讓使用者輸入一個字元，判斷它是包括大、小寫的英文字母（alphabet）、數字（number）、或者其它字元（symbol）。例如：a為英文字母、9為數字、$為其它字元。
### 範例輸入1
```shell
P
```
### 範例輸出1
```shell
P is an alphabet.
```
### 範例輸入2
```shell
@
```
### 範例輸出2
```shell
@ is a symbol.
```
### 範例輸入3
```shell
7
```
### 範例輸出3
```shell
7 is a number.
```
