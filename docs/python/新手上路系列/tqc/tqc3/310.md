# TQC+ 程式語言Python 310 迴圈公式計算
題目說明：請使用迴圈敘述撰寫一程式，讓使用者輸入正整數n (1 < n)，計算以下公式的總和並顯示結果：

$$
\frac{1}{1 + \sqrt{2}} + \frac{1}{\sqrt{2} + \sqrt{3}} + \frac{1}{\sqrt{3} + \sqrt{4}} + \cdots + \frac{1}{\sqrt{n-1} + \sqrt{n}}
$$

 
$$ 11+2+12+3+13+4+…+1n−1+n $$

> 提示：輸出結果至小數點後四位。
### 範例輸入
```shell
8
```
### 範例輸出
```shell
1.8284
```
