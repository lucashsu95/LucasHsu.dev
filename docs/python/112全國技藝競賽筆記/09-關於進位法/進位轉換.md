# 進位轉換 - 進位法

## 其他進位制轉換為十進位制
使用畫格子的方式

## 十進位制轉換為其他進位制
使用短除法

![](./關於進位法.png)

## 範例1
將二進制數值1101001轉換為十六進制，其值為多少？<br>
首先，將1101001透過畫格子的方式轉換為十進制，得到105。<br>
接著，使用短除法，將105除以16。<br>
(除數為16是因為要轉換的進制是十六進制)<br>
答案：69

## 範例2
英文字母「A」的十進制ASCII值為65，則字母「Q」的十六進制ASCII值為多少？<br>
首先，計算出Q的十進制ASCII值為81。<br>
接著，使用短除法，將81除以16。<br>
(除數為16是因為要轉換的進制是十六進制)<br>
答案：51
